# VV Hybrid Universe Manifesto

Foundational vision and philosophy for the `VV Hybrid Universe`.

## What This Repository Is

This repository holds the worldview behind the ecosystem.

It is not the system core.
It is not the invention repository.
It is the place where the direction, principles and long-term meaning of the ecosystem are stated clearly.

## Public Role In The Ecosystem

- `VV Hybrid Universe` is the ecosystem
- `vv-hybrid-univers-manifesto` is the vision layer
- `vv-hi-framework` is the invention layer
- `vv-nexus` is the system core
- `LEA`, `VNEX`, `VV Pulse`, and related products are product surfaces

## Core Idea

VV Hybrid Universe is being built around a local-first belief:

- intelligence should stay close to the user
- memory should grow more personal over time
- the user should control what stays private, what becomes hybrid, and what becomes public
